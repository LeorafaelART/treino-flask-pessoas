# Sistema de Pessoas

Projeto simples desenvolvido com Flask para estudo de:

- Flask
- HTML
- CSS
- SQLite
- DBeaver
- Git/GitHub

## Funcionalidades

- Cadastro de pessoas
- Pesquisa por nome, CPF e ID
- Exclusão de registros
- Contador de registros
- Integração com banco SQLite

## Tecnologias utilizadas

- Python
- Flask
- SQLite
- HTML
- CSS
